Technical Comment

This program finds factors using Pollard's rho algorithm, which is faster than the standard method of trial division. However, it is not the fastest algorithm available (General number field sieve), and quantum computers will be able to factor numbers even faster. In practice, this program can factor all numbers up to 1024 in a second.
